# 사용자 시나리오 (User Scenarios): TimeFilm

*본 문서는 [사용자는 __하기 위해 __한다] 형식을 준수하여 TimeFilm의 핵심 기능 플로우를 시나리오 형태로 정의한 문서입니다.*

## 1. 방문자 (Visitor): TimeFilm 페이지 방문 및 상호작용 과정

- 사용자는 **인플루언서의 다양한 소셜 미디어 채널과 포트폴리오를 한 곳에서 확인하기 위해** **인스타그램이나 프로필에 기재된 유일한 URL(`timefilm.com/@username`)을 클릭한다**.
- 사용자는 **페이지 소유자가 어떤 사람인지 파악하기 위해** **화면 최상단에 렌더링된 프로필 이미지, 닉네임, 그리고 한 줄 자기소개(Bio)를 읽어본다**.
- 사용자는 **소유자가 업로드한 최신 작업물이나 쇼핑몰로 바로 이동하기 위해** **화면 중앙에 수직으로 나열된 링크 버튼들 중 원하는 텍스트가 적힌 버튼을 탭(클릭)한다**.
- 사용자는 **자신이 둘러보고 있는 유용한 TimeFilm 프로필을 타인에게도 전달하기 위해** **페이지 하단이나 우측 상단에 위치한 '공유하기(Share)' 아이콘을 클릭하여 URL을 클립보드에 복사한다**.

## 2. 소유자 (Owner): 링크 추가/수정/삭제 등 관리 과정

### 2.1. 링크 추가(Create) 및 정렬 과정
- 사용자는 **새롭게 촬영한 유튜브 영상이나 진행 중인 이벤트 주소를 팬들에게 즉시 홍보하기 위해** **TimeFilm 관리자 대시보드에 구글(Google) 혹은 깃허브(GitHub) 계정으로 간편 로그인한다**.
- 사용자는 **입력 폼이 위치한 텅 빈 링크 카드 슬롯을 생성하기 위해** **대시보드 상단에 있는 눈에 띄는 '새 링크 추가' 버튼을 클릭한다**.
- 사용자는 **방문자가 해당 버튼을 클릭했을 때 어디로 렌딩되는지 직관적으로 알게 하기 위해** **생성된 카드의 제목(Title) 필드에 "최신 브이로그 보러가기"라고 타이핑한다**.
- 사용자는 **버튼 클릭 시 올바른 웹페이지로 정상 접속되게 하기 위해** **제목 필드 하단에 위치한 주소(URL) 필드에 복사해 둔 유튜브 영상 링크 주소를 붙여넣기한다**.
- 사용자는 **방금 입력한 링크 버튼이 실제 모바일 화면에서 잘 어울리는지 시각적으로 검증하기 위해** **대시보드 우측 영역에 마련된 '실시간 모바일 렌더링 프리뷰(미리보기)'를 살펴본다**.
- 사용자는 **추가한 신규 링크가 방문자들에게 가장 먼저 노출되게 하기 위해** **해당 링크 카드 좌측에 위치한 핸들(그립) 아이콘을 마우스로 꾹 누른 채 리스트의 최상단으로 드래그 앤 드롭한다**.

### 2.2. 링크 내용 수정(Update) 및 노출 상태 변경 과정
- 사용자는 **과거에 입력한 링크 제목의 오타나 URL 변경사항을 바로잡기 위해** **원하는 카드의 텍스트 입력 폼을 그대로 클릭하여 내용을 다시 타이핑한다**.
- 사용자는 **기간이 모두 종료된 한정 판매 상품 링크 버튼을 방문자 화면에서 잠시 감추기 위해** **비활성화 목적 하에 해당 링크 카드 우측 여백에 있는 토글 스위치를 마우스로 한 번 클릭하여 'OFF' 상태로 스위칭한다**.

### 2.3. 링크 삭제(Delete) 과정
- 사용자는 **더 이상 유효하지 않거나 영구적으로 사용할 일이 없어진 예전 포트폴리오 링크를 목록에서 아예 없애기 위해** **해당 링크 카드 우측 하단에 위치한 '휴지통 모양 아이콘(삭제)'을 클릭한다**.
- 사용자는 **순간적인 조작 실수로 소중한 링크 데이터가 날아가는 것을 방지하기 위해** **화면에 출력된 경고 모달(Modal) 창의 "정말로 이 링크를 영구 삭제하시겠습니까?" 문구를 읽은 후 최종적으로 붉은색 '삭제' 버튼을 클릭한다**.
