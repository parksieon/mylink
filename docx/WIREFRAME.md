# 와이어프레임 (Wireframe): 마이링크(MyLink)

본 문서는 아스키 아트(ASCII Art) 텍스트를 기반으로 작성된 마이링크 클론 서비스의 핵심 두 가지 화면(퍼블릭 모바일 뷰, 관리자 대시보드 뷰) 기획안입니다.

## 1. 퍼블릭 모바일 뷰 (방문자 공유 화면)
방문자가 고유 주소(예: `mylink.com/@username`)로 접속했을 때 렌더링되는 최종 모바일 프로필 화면입니다. 모바일 기기의 세로 비율에 최적화되어 표시됩니다.

```text
+---------------------------------------+
|                                  [공유]|
|                                       |
|             [ 프로필 사진 ]           |
|                  (당)                 |
|                                       |
|               @Nickname               |
|    안녕하세요, 크리에이터 아무개입니다. |
|    비즈니스 문의는 디엠으로 주세요!   |
|                                       |
|  +---------------------------------+  |
|  |     🎬 최신 브이로그 보러가기   |  |
|  +---------------------------------+  |
|                                       |
|  +---------------------------------+  |
|  |     📷 인스타그램 사진 보기     |  |
|  +---------------------------------+  |
|                                       |
|  +---------------------------------+  |
|  |     🛍 스토어 (한정 판매!)      |  |
|  +---------------------------------+  |
|                                       |
|        (트위터) (페북) (유튜브)       |
|                                       |
|            powered by MyLink          |
+---------------------------------------+
```

## 2. 관리자 대시보드 뷰 (소유자 편집 화면 / 데스크톱 기준)
앱 소유자가 링크를 추가하고 수정할 수 있는 어드민 페이지입니다. 화면 좌측은 데이터를 입력/수정하는 편집 패널이고, 우측은 실시간으로 결과를 보여주는 화면으로 분할됩니다.

```text
+-----------------------------------------------------------------------------------+
|  [MyLink 로고]    링크 편집    테마 디자인    통계 데이터            [프로필 관리]|
+-----------------------------------------------------------------------------------+
|                                               |                                   |
|  [ 프로필 설정 영역 ]                         |  [ 실시간 미리보기 (Live Preview) ]|
|  +-----------------------------------------+  |                                   |
|  | (이미지) | 닉네임: @Nickname            |  |      +---------------------+      |
|  | 업로드   | 소개글: 안녕하세요,...       |  |      |   @Nickname         |      |
|  +-----------------------------------------+  |      |   안녕하세요,...    |      |
|                                               |      |                     |      |
|  [ + 새 링크 추가하기 ] 버튼 (클릭 시 폼 생성)|      |  +---------------+  |      |
|                                               |      |  | 최신 브이로그  |  |      |
|  [ 링크 관리 리스트 (Drag & Drop) ]           |      |  +---------------+  |      |
|  ===========================================  |      |                     |      |
|  = ::: | 제목: 최신 브이로그 보러가기   | [ON] |      |  +---------------+  |      |
|  = ::: | URL : youtube.com/...          |[삭제]|      |  | 인스타그램    |  |      |
|  ===========================================  |      |  +---------------+  |      |
|                                               |      |                     |      |
|  ===========================================  |      |  +---------------+  |      |
|  = ::: | 제목: 인스타그램 사진 보기     | [ON] |      |  | 스토어        |  |      |
|  = ::: | URL : instagram.com/...        |[삭제]|      |  +---------------+  |      |
|  ===========================================  |      |                     |      |
|                                               |      |     (SNS 아이콘)    |      |
|  ===========================================  |      |                     |      |
|  = ::: | 제목: 스토어 (한정 판매!)      |[OFF]|      +---------------------+      |
|  = ::: | URL : shop.com/...             |[삭제]|                                   |
|  ===========================================  |                                   |
|                                               |                                   |
+-----------------------------------------------------------------------------------+
```

### 💡 주요 UI 컴포넌트(아스키 요소) 설명
- **`:::` (좌측 핸들 기호)**: 마우스로 꾹 눌러 위아래로 당기면 각 카드의 노출 순서를 바꿀 수 있는 직관적인 **Drag & Drop 핸들(그립)** 영역입니다.
- **`[ON]/[OFF]` 토글 버튼**: 링크 카드를 아예 삭제하지 않고도 퍼블릭 모바일 뷰(방문자 화면)에서 임시로 감추거나 다시 노출시킬 수 있는 **활성화 제어 스위치**입니다.
- **우측 실시간 미리보기 패널**: 사용자가 좌측에서 링크의 주소, 제목을 수정하거나 토글 상태를 비활성화(`[OFF]`)하면, 페이지의 새로고침 없이 즉각적으로 우측 가상의 모바일 프레임에 동일하게 내용이 치환되어 그려집니다.
