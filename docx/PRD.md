# 제품 요구사항 정의서 (PRD): 마이링크(MyLink)

## 1. 프로젝트 개요
- **프로젝트명:** 마이링크 (MyLink)
- **목적:** 수많은 온/오프라인 링크(SNS, 포트폴리오, 스토어 등)를 단 하나의 프로필 페이지로 통합하여 간편하게 공유하고 관리할 수 있도록 지원하는 멀티링크 서비스.
- **대상 사용자:** 
  - 다중 SNS 채널(인스타그램, 유튜브, 틱톡 등)을 운영하는 **인플루언서 / 크리에이터**
  - 개별 작품, 이력서, 이메일 연락망을 모아두고 싶은 **프리랜서 / 아티스트**
  - 온라인 쇼핑몰, 메뉴판, 예약 링크 등 고객 안내가 필요한 **소상공인 / 기업 브랜드**

## 2. 핵심 기능 목록 (Key Features)

### [필수 기능 (Must-Have)]
1. **사용자 계정 관리 (소셜 로그인 중심):** 구글(Google) 및 깃허브(GitHub) Oauth 기반의 간편 로그인 연동
2. **고유 프로필 주소 할당:** `/@[username]` 형식의 고유 URL 선점
3. **프로필 설정:** 프로필 이미지, 닉네임, 한 줄 자기소개(Bio) 작성
4. **링크 관리 시스템 (CRUD):** 
   - 무제한 링크 아이템 생성, 수정, 삭제 기능
   - 각 링크의 노출 상태 활성/비활성화 토글
5. **드래그 앤 드롭 (Drag & Drop) 정렬:** 마우스 조작을 통한 링크 순서 직관적 변경
6. **동적 사용자 통계 제공 (핵심 가치):** 내 프로필 총 방문자 수 집계, 생성된 개별 링크에 대한 누적 클릭 횟수(Click) 추적
7. **라이브 프리뷰 (Live Preview):** 대시보드 편집 시 우측에 모바일 화면 기반의 실시간 결과물 미리보기 제공
8. **퍼블릭 모바일 뷰:** 실제 방문자에게 공유되고 렌더링되는 모바일 퍼스트 프로필 페이지 제공

### [선택 기능 (Nice-to-Have)]
1. **테마 커스터마이징:** 배경색, 버튼 둥글기, 텍스트 색상, 그리고 프리셋 테마(다크모드 등) 선택 지원
2. **블록 타입 다양화:** 일반 링크 텍스트 버튼 외에 소셜 아이콘 바(Bar) 전용 링크 지원

## 3. 각 기능 상세 설명

### 3.1. 사용자 인증 & 프로필 설정
- 가입 허들은 타겟 사용자에 친숙한 **구글 및 깃허브 로그인**을 연동하여 최소화합니다 (우선 비밀번호가 없는 간편 로그인으로 접근 방식 통일). 유료 요금제 구조가 없이 무료 모델로 구성됩니다.
- 사용자가 원하는 텍스트로 `mylink.com/@아이디` 형식의 고유 주소를 선점할 수 있습니다. 이미 배포된 이름은 중복 경고 밸리데이션이 표시됩니다.
- 초기 대시보드에서 아바타 이미지, 표시할 이름, 자신을 표현하는 텍스트 블록(Bio)을 입력하며 이는 접속자 상단 뷰에 고정으로 렌더링됩니다.

### 3.2. 링크 관리 시스템 & UI
- `새 링크 추가` 버튼을 클릭할 때마다 제목(Title)과 연결할 대상 주소(URL)를 입력받는 폼 컴포넌트가 위쪽에 추가됩니다.
- 각 카드의 그립(핸들) 영역을 마우스로 잡고 드래그하면 순서가 즉시 치환되고 DB에 반영됩니다.
- 특정 프로모션이나 이벤트 링크의 경우, 삭제할 필요 없이 '보이기/숨기기 토글'을 눌러 퍼블릭 뷰에서 임시 비활성화 처리가 가능합니다.

### 3.3. 데이터 통계 시스템
- 사용자가 자신의 대시보드 메인 혹은 별도 탭에서 직관적인 수치를 확인할 수 있습니다.
- "내 프로필 방문 횟수"의 증가 추이와 사람들이 "어떤 버튼을 얼마나 클릭했는지"에 대한 핵심 트래킹을 제공합니다. 이는 크리에이터 등 유저들의 성과 측정을 위해 중요하게 사용됩니다.

### 3.4. 라이브 프리뷰 인터페이스
- 사용자가 DB를 업데이트할 때마다(프로필 수정, 링크 추가, 테마 변경 등) 우측 가상의 모바일 렌더링 영역에서 완성된 페이지 형태가 실시간 동기화되어 나타납니다.
