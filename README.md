# Linktree Clone Project

이 프로젝트는 다수의 소셜 링크를 통합 관리할 수 있는 멀티링크 프로필 서비스입니다.
사이버 펑크/네오브루탈리즘 포트폴리오 프로젝트에서 **링크트리 클론** 프로젝트로 초기화되었습니다.

## Getting Started

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
